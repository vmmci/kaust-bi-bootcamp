 # KAUST Bioinformatics Bootcamp Portfolio

This repository contains my work from the KAUST Bioinformatics Bootcamp, covering computational biology foundations, R, Python, Linux, and biological data analysis workflows.

The work includes practice notebooks, scripts, and reports related to exploratory data analysis, visualization, shell scripting, RNA-seq analysis, and single-cell RNA-seq preprocessing.

## Repository Structure

    kaust-bi-bootcamp/
    ├── Bio/
    ├── Linux/
    ├── Python/
    └── R/

## Main Areas Covered

### Bioinformatics

The `Bio` folder includes bioinformatics-related reports and analysis files, including work on bulk RNA-seq and single-cell RNA-seq workflows.

Topics include:

- Biological data handling
- Metadata organization
- Bulk RNA-seq analysis
- Single-cell RNA-seq preprocessing
- Cell type identification concepts
- Reproducible analysis using R Markdown and HTML reports

### R

The `R` folder includes R-based case study work focused on data analysis and statistical reasoning.

Topics include:

- Data cleaning and transformation
- Exploratory data analysis
- Correlation and covariance analysis
- Principal Component Analysis (PCA)
- Statistical testing
- Visualization and reporting

### Python

The `Python` folder includes practice notebooks for data analysis and visualization.

Topics include:

- Exploratory Data Analysis (EDA)
- Pandas workflows
- Data visualization
- Basic data processing using Python notebooks

### Linux

The `Linux` folder includes command-line and shell scripting practice.

Topics include:

- Linux file system navigation
- Bash scripting
- File and directory management
- Backup automation using shell scripts

## Skills Demonstrated

- Bioinformatics workflow understanding
